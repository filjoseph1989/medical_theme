How to add map in mmfc theme?
  1. You need the file
      functions.php - add google hosted api jquery and add map.js in the footer
      js/maps.js - code the handle the map display
      footer.php - has a <div id="map"></div> wherein the display happens.
  2. You need a coordinate from google map.
      Use search to locate the address,
      right click and choose "direction here"
      in the url, look for coordinates, like (7.087, 125.63193).
  3. Create a javascript code in maps.js
